Required icon sizes for PWA:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- maskable-icon.png (should have padding around the logo for safe area)
- apple-touch-icon.png (180x180px)

To generate these icons:
1. Use your existing logo image file
2. Use an online tool like https://app-manifest.firebaseapp.com/ or https://maskable.app/editor
3. Place all generated icons in this /icons/ directory
4. For the maskable icon, ensure your logo has padding around it (safe area)

For optimal results:
- Use a PNG format with transparency
- Make sure your logo is centered
- Use a square image as source
